API Documentation

Pwned Passwords API

GET /pwned/range/prefix

Similiar to the Pwned Passwords API from Have I Been Pwned, but without numbers. Without numbers the lines have fixed length, allowing faster searching on the server.

Storage API

Some key-value storage. Keys are composed of a 20 character user identifier and a path. All user data is deleted if not accessed for 20 minutes. The client stores encrypted files here.

PUT /storage/user_key/path/without/trailing/slash

store the request body at this path

DELETE /storage/user_key/path/without/trailing/slash

delete stored data

GET /storage/user_key/path/without/trailing/slash

retrieve stored data

GET /storage/user_key/
GET /storage/user_key/path/with/trailing/slash/

directory listing as HTML. The selector #ls li a finds the links containing the relative paths to stored entries.

Client Application

GET /client/path

The responsive security client application, written in ECMAScript 2017. To use it, the cookie product_key containing the product key is required. The error message contains a form to set the cookie.

GET|POST /client/gameserver_retrieve_productkey

Special CTF endpoint for the checker to gain access to the product key.
